## MySQL 데이터베이스 생성

```sql
CREATE DATABASE mydb;
--- 아이디 및 패스워드 설정
CREATE USER 'myuserid'@'%' IDENTIFIED BY 'mypassword';
GRANT ALL ON mydb.* TO 'myuserid'@'%';
FLUSH PRIVILEGES;
```

`mydb`: 데이터베이스 이름

`myuserid`: 사용자 id

`mypassword`: 사용자 패스워드

### 사용자 패스워드가 생각나지 않을 때

루트 사용자로 로그인 후 일반 사용자 패스워드는 쉽게 변경 가능

```sql
SET PASSWORD FOR 'myuserid'@'%'='new_password';
FLUSH PRIVILEGES;
```
