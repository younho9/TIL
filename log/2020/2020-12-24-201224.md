## Github에 있는 Pull Request를 Local로 가져오기

```bash
git fetch origin pull/{풀 리퀘스트 ID}/head:{브랜치 이름}
git checkout {브랜치 이름}
```

### 업데이트된 Pull Request 가져오기

```bash
git pull origin pull/{풀 리퀘스트 ID}/head
```
