## 모듈 전체 목록 보기

```bash
pip list
```

## 업데이트가 필요한 항목 보기

```bash
pip list --outdated
```

## 업데이트가 필요하지 않은 최신 항목 보기

```bash
pip list --uptodate
```

## PIP 업데이트하기

```bash
pip install --upgrade pip
```
